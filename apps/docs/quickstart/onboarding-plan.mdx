## Overview

A plan is a representation of a package in the SaaS world. Creating your first plan in Unprice, involves creating an account and setting up your first project. The following video shows how to set up your project and publish your first plan.

<AccordionGroup>
  <Accordion title="Create account & project">
    Sign up into [Unprice](https://app.unprice.dev)
  </Accordion>
  <Accordion title="Set up your plan">
    A plan represent a package of features that you are offering. Every plan can have multiple versions so you can iterate on different features for the same plan.
  </Accordion>
  <Accordion title="Create a new plan version">
    A plan version represent a version of the plan with different configurations, like billing frequency, payment provider, currency, etc. Once a plan version is published, customer can subscribe to it. Plan versions are append only, meaning after they are published you can't change them. See why [here](/a)
  </Accordion>
  <Accordion title="Add Features">
    Every plan version contains a set of features that you can configure according to your needs. Flat features, usage based features or quatity based features and more.
  </Accordion>
  <Accordion title="Review & publish">
    Review the pricing card and publish the new plan version.
  </Accordion>
</AccordionGroup>
